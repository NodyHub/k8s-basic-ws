# Container to deployment

## List All

```
kubectl get all
```

## List Pods

```
kubectl get pods
```

## List Deployments

```
kubectl get deployments
```

## Create Deployment

```
kubectl create deployment k8s-ws --image=nodyd/k8s-ws
```

## List Deployments

```
kubectl get deployments
```

## List pods

```
kubectl get pods
```

## Delete pod

```
kubectl delete pod/<name from previous output>
```

## List pods

```
kubectl get pods
```
